Tabla comparativa de propiedades CSS

Propiedad Ejemplo visual Descripción
background Fondo azul claro Define el fondo del elemento.
color Texto en rojo oscuro Define el color del texto.
padding Caja con más espacio interno Espacio entre el contenido y el borde.
text-align Texto alineado a la derecha Alineación del texto (izquierda, centro, derecha, justificado).
text-decoration Texto subrayado Agrega decoraciones: underline, line-through, overline.
border Borde punteado verde Define el grosor, estilo y color del borde.
border-radius Caja con esquinas redondeadas Redondea las esquinas del borde.
